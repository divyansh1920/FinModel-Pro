## рҹҸҰ What is FinModel Pro?

> *"The exact analytical framework used in analyst training programs at top investment banks вҖ” now open source."*

**FinModel Pro** is a fully self-contained, single-file financial modelling toolkit that runs directly in your browser вҖ” no server, no installations, no subscriptions. Just download the HTML file and double-click.

Built for **students, aspiring analysts, and finance professionals** who want to learn financial modelling the way it's actually done at firms like Goldman Sachs, JPMorgan, Morgan Stanley, BlackRock, Lazard, and Evercore вҖ” by working with real models, not watching videos.

Every model is **live and automated** вҖ” enter your assumptions and every output recalculates instantly.

## рҹ“җ Formula Reference

Every model uses institutional-standard formulas. Here's a snapshot:

<br/>

**DCF вҖ” Free Cash Flow to Firm**
```
FCFF  =  NOPAT  +  D&A  вҲ’  CapEx  вҲ’  О”NWC
      =  [EBIT Г— (1 вҲ’ t)]  +  D&A  вҲ’  CapEx  вҲ’  О”NWC
```

**DCF вҖ” Terminal Value (Perpetuity Growth)**
```
TV  =  FCFFвӮ… Г— (1 + g)
       в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
          WACC  вҲ’  g
```

**WACC вҖ” Weighted Average Cost of Capital**
```
WACC  =  (E/V Г— Ke)  +  (D/V Г— Kd Г— (1 вҲ’ t))

where:
  Ke  =  Rf  +  ОІ Г— ERP          [CAPM]
  Kd  =  Pre-Tax Cost of Debt
  E/V =  Equity / (Equity + Debt)
```

**LBO вҖ” Internal Rate of Return**
```
0  =  ОЈ  CFвӮң / (1 + IRR)бө—       [solved via Newton-Raphson]
      t=0

MOIC  =  Exit Equity Value / Entry Equity Contribution
```

**Bond вҖ” Price & Duration**
```
Price  =  ОЈ  C / (1+y)бө—  +  F / (1+y)бҙә
          t=1

Macaulay Duration  =  ОЈ [t Г— PV(CFвӮң)] / Price
Modified Duration  =  Macaulay Duration / (1 + y/n)
DV01               =  Modified Duration Г— Price Г— 0.0001
```

**M&A вҖ” EPS Accretion / Dilution**
```
Pro-Forma NI   =  Acquirer NI  +  Target NI  +  Synergies  вҲ’  Financing Cost  вҲ’  Fees
Pro-Forma EPS  =  Pro-Forma NI / Pro-Forma Shares Outstanding
Accretion/(Dilution)  =  Pro-Forma EPS  вҲ’  Standalone EPS
```

> рҹ“ҡ All 80+ formulas are documented inside the **Formula Glossary** page of the app itself.

- Input: 5-year revenue projections, EBIT margin, D&A, CapEx, NWC, WACC, terminal growth rate
- Output: FCFF per year, PV of FCFFs, Terminal Value, PV of TV, Enterprise Value, Equity Value, **Implied Share Price**
- Formula: `TV = FCFFвӮ… Г— (1 + g) / (WACC вҲ’ g)`

**Exit Multiple Method**
- Same structure, terminal value via `TV = EBITDA Year 5 Г— Exit EV/EBITDA Multiple`
- **Method comparison card** shown when both tabs are completed: Price variance in $ and %

**Bear / Base / Bull Scenarios**
- Three independent scenario columns with their own WACC and g/multiple inputs
- All three implied prices calculated simultaneously
- Auto-commentary: *"Base: $XX. Bear downside: вҲ’XX%. Bull upside: +XX%"*

**Sanity Checks (live)**
- вҡ  Terminal growth rate exceeds WACC
- вҡ  Terminal Value > 90% of EV
- вҡ  All FCFFs are negative

**Inputs:** Entry EBITDA, entry multiple, debt tranches (Senior Secured, Senior Unsecured, Sub Debt вҖ” each with leverage multiple and interest rate), 5-year operating assumptions, exit year, exit multiple.

```javascript
// IRR via Newton-Raphson вҖ” implemented in pure JS
function calcIRR(cashflows) {
  let rate = 0.20;
  for (let i = 0; i < 1000; i++) {
    let npv = 0, dnpv = 0;
    cashflows.forEach((cf, t) => {
      npv  += cf / Math.pow(1 + rate, t);
      dnpv -= t * cf / Math.pow(1 + rate, t + 1);
    });
    const newRate = rate - npv / dnpv;
    if (Math.abs(newRate - rate) < 1e-8) break;
    rate = newRate;
  }
  return isFinite(rate) ? rate : null;
}
```

- IRR > 25% = green | 20вҖ“25% = amber | < 20% = red
